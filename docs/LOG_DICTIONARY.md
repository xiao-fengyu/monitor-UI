## 🔴 严重级别（需立即关注）

### SSH 登录失败
| 关键词 | `Failed password for` |
|--------|----------------------|
| 含义 | 有人尝试通过 SSH 登录服务器，但密码错误 |
| 严重性 | 如果是已知用户的操作，可能是误操作；如果是陌生 IP，可能是暴力破解攻击 |
| 建议 | 检查 IP 来源，频繁失败可考虑封禁 |

### SSH 认证失败（PAM）
| 关键词 | `pam_unix(sshd:auth): authentication failure` |
|--------|-----------------------------------------------|
| 含义 | PAM 模块拒绝了认证请求 |
| 原因 | 密码错误、用户不存在、认证模块配置问题 |

### 无效用户名
| 关键词 | `Invalid user` |
|--------|---------------|
| 含义 | 有人尝试使用不存在的用户名登录 |
| 严重性 | 通常是自动化扫描工具在爆破 |

### 磁盘 I/O 错误
| 关键词 | `I/O error` |
|--------|------------|
| 含义 | 读写磁盘时发生硬件级错误 |
| 严重性 | 🔴 **最高优先级** |
| 建议 | 立即检查硬盘健康状况（`smartctl`），准备数据备份 |

### 磁盘空间不足
| 关键词 | `No space left on device` |
|--------|--------------------------|
| 含义 | 磁盘已满，无法写入新数据 |
| 建议 | 清理无用文件、日志、Docker 镜像等 |

### 任务/服务失败
| 关键词 | `Failed` / `failed` |
|--------|---------------------|
| 含义 | 某个任务或服务执行失败 |
| 建议 | 查看完整日志了解具体失败原因 |

## 🟡 警告级别（建议了解）

### SSH 登录成功
| 关键词 | `Accepted password for` |
|--------|------------------------|
| 含义 | 用户通过 SSH 成功登录 |
| 备注 | 正常行为，但如果来自未知 IP 需关注 |

### API 限流
| 关键词 | `429` / `Too Many Requests` |
|--------|-----------------------------|
| 含义 | 模型调用频率过高，被服务端限流 |
| 影响 | 请求被拒绝，稍后自动恢复 |
| 建议 | 降低请求频率，或使用缓存 |

### 请求超时
| 关键词 | `TimeoutError` |
|--------|---------------|
| 含义 | 请求响应时间超过阈值 |
| 原因 | 网络拥堵、目标服务响应慢、连接池耗尽 |

### 服务停止
| 关键词 | `Stopped` / `Stopping` |
|--------|----------------------|
| 含义 | 某个 systemd 服务正在关闭或已关闭 |
| 注意 | 如果是非预期停止，需排查原因 |

### 网络断开
| 关键词 | `WebSocket disconnected` |
|--------|-------------------------|
| 含义 | 长连接断开 |
| 原因 | 网络波动、服务重启、超时 |
| 影响 | 通常会自动重连，短暂中断无影响 |

## 📊 OpenClaw 特定日志

### 网关日志
| 关键词 | `openclaw.*gateway` |
|--------|--------------------|
| 含义 | OpenClaw 网关服务的运行日志 |
| 关注点 | 启动、重启、错误、模型调用状态 |

### 模型限流
| 关键词 | `429` + `DeepSeek` |
|--------|-------------------|
| 含义 | DeepSeek 模型调用被限流 |
| 影响 | 该次请求失败，系统会尝试使用其他模型或重试 |

### A2A 任务超时
| 关键词 | `A2A task timeout` |
|--------|-------------------|
| 含义 | Agent 间协作任务超时 |
| 原因 | 对方 Agent 的模型响应慢或卡住 |
| 影响 | 任务失败，需要重试或人工介入 |

### 记忆筛选失败
| 关键词 | `filterRelevant failed` |
|--------|----------------------|
| 含义 | 记忆相关性筛选失败 |
| 原因 | 模型不可用或网络问题 |

### 回退到空结果
| 关键词 | `fallback to empty` |
|--------|-------------------|
| 含义 | 没有可用的候选结果，返回空数据 |
| 影响 | 依赖该数据的功能会显示为空 |
