## ✨ 功能概览

### 日志聚合
- 实时采集 openclaw-gateway 和 searxng 日志
- 中文含义解释（看不懂日志也能明白发生了什么）
- 按级别/时间/来源筛选
- 日志趋势可视化

### 监控告警
- 服务存活监控（Gateway / SearXNG）
- 错误频率告警
- 模型可用性监控
- 磁盘/内存/CPU 资源监控
- 告警规则可配置

### 备份管理
- 定时自动备份配置与数据到 GitHub
- 手动一键备份
- 备份历史查看
